# portfolio-weight-optimization
A portfolio optimization tool using Monte Carlo simulations to determine optimal asset weights. It evaluates thousands of random allocations to identify the Efficient Frontier and maximize the Sharpe Ratio for risk-adjusted returns.


Methodology: Implemented Monte Carlo simulations to infer the most likely risk/return profile of a basket of securities.

Mathematical Insight: Calculated covariance matrices to account for inter-asset correlations.

Outcome: Visualized the Efficient Frontier and identified the "Max Sharpe" portfolio for a given risk appetite.
